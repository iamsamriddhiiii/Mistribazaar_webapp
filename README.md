## MistriBaazar

MistriBaazar is a web-based service marketplace that connects customers with skilled workers such as electricians, plumbers, carpenters, and other local service providers through a single platform.

---

##  Project Overview

MistriBaazar is designed to simplify the process of finding and hiring skilled workers. Both customers and workers register using a common authentication system. Based on their role, users are redirected to their respective dashboards where they can perform role-specific actions.

The platform aims to bridge the gap between service seekers and service providers in an efficient and user-friendly manner.

---

##  Features

- Single registration and login system for both customers and workers  
- Role-based dashboards (Customer / Worker)  
- Workers can add and manage their service details  
- Customers can browse worker listings  
- Hire request system for customers  
- Secure session handling and authentication  

---

##  Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Nodejs  
- **Database:** MySQL  
- **Tools:** VS Code, GitHub, XAMPP
